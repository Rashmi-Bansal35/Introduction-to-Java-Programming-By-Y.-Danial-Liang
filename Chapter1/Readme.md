## 🛠️ How to Run Java Code

1. **Open Terminal or Command Prompt**
2. **Navigate to the chapter folder:**
   ```bash
   cd chapter1
   ```
3. **Compile a Java file:**
   ```bash
   javac HelloWorld.java
   ```
4. **Run the program:**
   ```bash
   java HelloWorld
   ```
