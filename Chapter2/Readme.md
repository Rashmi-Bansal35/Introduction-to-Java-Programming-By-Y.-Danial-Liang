# 🌟 Chapter 2: Elementary Programming

Welcome to **Chapter 2** of the companion repository for *Introduction to Java Programming* by **Y. Daniel Liang**! 🚀

This chapter expands on variables, primitive data types, expressions, operators, and essential console input/output mechanics (`Scanner` class). These concepts are the structural pillars of programmatic problem-solving.

---

## 🚀 What You'll Learn

* 🧮 **Variables and Data Types:** Declaring variables, primitive types (`int`, `double`, `long`), and casting rules.
* ➕ **Arithmetic Operations:** Operators (`+`, `-`, `*`, `/`, `%`), precedence rules, and expression evaluation.
* 🖨️ **Console Input and Output:** Using `Scanner` to read user data and formatting displays.
* 📝 **Assignment & Shorthands:** Understanding increment/decrement (`++`, `--`) and shorthand assignment operators.
* 🧩 **Problem Solving with Java:** Translating algebraic, financial, and physics formulas into robust execution steps.

# 🛠️ How to Compile & Run Java Programs
If you are using a terminal or command prompt, follow these simple steps to compile and run any Java program.

## 1️⃣ Navigate to the Chapter Folder
```bash
cd chapter2
```

## 2️⃣ Compile the Java File
Before execution, Java source code (`.java`) must be compiled into bytecode (`.class`).
```bash
javac Question1.java
```

## 3️⃣ Run the Compiled Class

While running the program, do not include the `.java` or `.class` extension.
```bash
java Question1
```

# 💡 Pro-Tip for Beginners

Java is **case-sensitive**!

`Main` and `main` are completely different things.

Always double-check:

- Capital letters  
  *(e.g., `System`, `String`)*
- Semicolons `;`
- Curly braces `{ }`
- Quotation marks `" "`

If you face compilation errors.
