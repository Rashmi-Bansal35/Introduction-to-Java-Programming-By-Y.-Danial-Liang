# 🌟 Chapter 5: Loops

Welcome to **Chapter 5** of the _Introduction to Java Programming_ by **Y. Daniel Liang**!  
This chapter is centered on loops, one of the most important tools for writing repeated tasks and processing data in Java.

---

## 🚀 What You'll Learn

- 🔁 **For Loops:**  
  Use `for` loops to run code a fixed number of times and iterate through counters.
- 🔄 **While Loops:**  
  Use `while` loops to repeat actions while a condition remains true.
- 🧩 **Do-While Loops:**  
  Use `do-while` loops when the code should execute at least once before checking the condition.
- 🧵 **Nested Loops:**  
  Use loops inside loops to generate patterns, tables, and multi-dimensional output.
- ✋ **Loop Control:**  
  Use `break` to exit a loop early and `continue` to skip specific iterations.
- 🧠 **Problem Solving:**  
  Apply loops to sum values, count items, validate input, and build repeated output.

# 🛠️ How to Compile & Run Java Programs
If you are using a terminal or command prompt, follow these simple steps to compile and run any Java program.

## 1️⃣ Navigate to the Chapter Folder
```bash
cd chapter5
```

## 2️⃣ Compile the Java File
Before execution, Java source code (`.java`) must be compiled into bytecode (`.class`).
```bash
javac Question1.java
```

## 3️⃣ Run the Compiled Class

While running the program, do not include the `.java` or `.class` extension.
```bash
java Question1
```

# 💡 Pro-Tip for Beginners

Loops allow you to execute a block of code repeatedly without writing the same statements again and again.

Always double-check:

- Loop initialization
- Loop condition
- Increment/decrement statements
- Proper placement of braces `{ }`
- Avoiding infinite loops
